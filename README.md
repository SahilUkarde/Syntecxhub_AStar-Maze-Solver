# A* Maze Solver

## Overview

This project implements the A* (A-Star) Pathfinding Algorithm to find the shortest path between a start point and a goal point in a maze. The algorithm efficiently explores possible paths using heuristic functions and guarantees the optimal solution when an admissible heuristic is used.

## How A* Works

The A* algorithm evaluates nodes using:

f(n) = g(n) + h(n)

Where:

* g(n): Cost from start node to current node
* h(n): Estimated cost from current node to goal
* f(n): Total estimated cost

The algorithm always explores the node with the lowest f(n) value, making it both efficient and optimal.
